Alap Config Server (bun-sqlite) — API Demo

Interactive test page for the REST API. Server running at http://localhost:3000.

GET /configs

List all saved config names.

    GET /configs/:name

    PUT /configs/:name

    DELETE /configs/:name

    GET /search

    Search across all configs by tag, text, or regex.

    POST /cherry-pick

    Resolve an expression against a config, return only matching links.

    POST /query

    Server-side expression resolution. Optionally merge multiple configs.
