How this works

Each nav button triggers an hx-get that fetches an HTML fragment. htmx swaps the fragment into #content. The fragments contain <alap-link> web components that self-initialize via the custom element registry — no re-init call needed.

The Alap config and IIFE build are loaded once in the <head>. That's it. No bundler, no framework, no hydration.
