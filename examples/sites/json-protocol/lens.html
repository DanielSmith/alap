:json: + Lens

APIs with rich metadata rendered in Alap's detail-inspection panel. The lens auto-detects field types: strings become key-value rows, arrays become chips, numbers stay numbers.

Open Library

Book search with cover images constructed via template URL (https://covers.openlibrary.org/b/id/${cover_i}-M.jpg). Author names, publication year, and edition count in meta.

Search books: Bridges | Architecture | Jazz History

Library of Congress

General collection search with subjects as tags, date, contributor, and original format in meta. Demonstrates rich metadata without deep nesting.

Search the collection: Bridges | Photography | Jazz History

Two Sources, One Query

Compose Open Library and Library of Congress results with the | operator. Same search term, different APIs, merged results.

Bridges (Open Library + LOC)

Both sources feed the same lens. Each item carries meta.source = 'json' plus its own envelope metadata.

MusicBrainz

Album search by artist with custom User-Agent header (required by the MusicBrainz API). Artist from nested path (artist-credit[0].artist.name), tags extracted via tags[].name.

Albums by: Miles Davis | Nina Simone | Radiohead
