All Together

Menu, lightbox, and lens wired to the same dataset and coordinated by RendererCoordinator. Escape walks back through whichever renderers you've opened.

Three renderers, one dataset

Each trigger resolves the same expression; the selector class decides which renderer paints the result. The menu stays text-only here so the lightbox and lens carry the visual weight.

bridges (menu) · bridges (lightbox) · bridges (lens)

the tour (menu) · the tour (lightbox) · the tour (lens)

<a class="alap-menu"     data-alap-linkitems=".bridge">bridges (menu)</a>
