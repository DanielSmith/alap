Expression syntax

      

        
.tagAll items with this tag

        
itemIdA specific item by ID

        
@macroExpand a named macro

        
.a + .bIntersection (both tags)

        
.a | .bUnion (either tag)

        
.a - .bSubtraction (a without b)
