# hugo-alap

Hugo module for Alap — shortcode that emits `<alap-link>` web components with full expression language support.

## Install

Add the module to your `hugo.toml`:

```toml
[module]
  [[module.imports]]
    path = "github.com/DanielSmith/alap/integrations/hugo-alap"
```

Full documentation: [docs/integrations/hugo.md](../../docs/integrations/hugo.md)
